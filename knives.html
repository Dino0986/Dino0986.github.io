Sketchup, 3D Printing, and Knife Making

Using Sketchup for knife making can be really usefull for rapid prototyping. Especially if you have a 3d printer. It lets you see how it fits, if you like the shape, and it gives you a template to work with.



The design files for all the knives are available online for free here.

The entire process is pretty easy to set up, and you don't need really expensive tools to do it. Just make sure that you know how to make basic shapes and extrusions in sketchup. Tutorials are easy to follow and available online.


*This is by no means a complete guide, using common sense and your own problem solving skills is required to build one of your own.*


Components Needed:

- Computer
- Sketchup, Pro or Make
- Slicing software, such as Slic3r or Cura


Tools Needed:

- 3D Printer
- Glue
- (Optional) Sandpaper
- (Optional) Paracord



This is a weekend project. Knife making is a longer process, but 3D printing can really speed up the prototyping process.
