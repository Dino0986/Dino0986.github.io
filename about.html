About The Robotics Class

Introductory Robotics Engineering

In this course you will build and program robots using microcontrollers, motors and sensors. You will use an Arduino microprocessor as the brain of your robot, wire it to motors and sensors, and use C as the programming language. Through this process you will learn the basics of computer programing, electro-assembly and robotics. The programming and electronics learned in this course teach the introductory skills and concepts to create a very large range of applications such as video game programming, creating interactive art installations, building electronic musical instruments or automating your house.

CSE1110: STRUCTURED PROGRAMMING 1:
Students are introduced to a general programming environment in which they write simple structured algorithms and programs that input, process and output data, use some of the more basic operators and data types, and follow a sequential flow of control. Prerequisite: None

CSE1240: ROBOTICS PROGRAMMING 1:
Students use an appropriate Robot Control Language (RCL) to design, develop, implement and debug robotics programs that employ standard structured programming constructs and simple data structures. In the process, they develop a general understanding of robots and the robotics environment. Prerequisite: CSE1110: Structured Programming 1

ELT1010: ELECTRO-ASSEMBLY 1:
Students apply basic fabricating and servicing techniques to construct and test electronic and electromagnetic devices and cables. Prerequisite: None

ELT1130: ROBOTICS 1:
Students apply the fundamentals of robotics systems and basic robotics functions. Prerequisite: None

ELT1140: ROBOTICS APPLICATIONS: Students apply the fundamentals of robotics systems and basic robotics functions. Prerequisite: ELT1010: Electro-assembly 1
